## PASO 2: Subir el Código

### Opción A: Desde la Terminal (si tiene acceso al ZIP descomprimido)

```bash
# Descomprimir el archivo
unzip cootracovi-disciplinarios.zip -d cootracovi-disciplinarios
cd cootracovi-disciplinarios

# Inicializar Git
git init
git add .
git commit -m "Initial commit - Sistema de Procesos Disciplinarios COOTRACOVI"

# Conectar con GitHub (reemplazar USUARIO por el nombre de usuario real)
git remote add origin https://github.com/USUARIO/cootracovi-disciplinarios.git
git branch -M main
git push -u origin main
```

### Opción B: Subir desde la Interfaz Web de GitHub

1. En el repositorio creado, hacer clic en "uploading an existing file"
2. Arrastrar todos los archivos del ZIP descomprimido
3. En "Commit changes", escribir: "Initial commit"
4. Hacer clic en "Commit changes"

## PASO 3: Configurar Vercel

1. Ir a https://vercel.com y entrar con la cuenta de GitHub
2. Hacer clic en "Add New..." → "Project"
3. Seleccionar el repositorio "cootracovi-disciplinarios"
4. Hacer clic en "Import"
5. Configurar las variables de entorno:

   Hacer clic en "Environment Variables" y agregar:

   | Variable | Valor |
   |---------|-------|
   | DATABASE_URL | mysql://usuario:password@servidor:3306/disciplinarios |
   | NEXTAUTH_SECRET | (generar con: openssl rand -base64 32) |
   | NEXTAUTH_URL | https://su-app.vercel.app |
   | NEXT_PUBLIC_BASE_URL | https://su-app.vercel.app |

6. Hacer clic en "Deploy"

## PASO 4: Ejecutar Migraciones (Importante)

Después de que Vercel termine el despliegue, ejecutar migraciones:

1. En Vercel, ir a la pestaña "Deployments"
2. Hacer clic en el último despliegue
3. Ir a "Functions" → Abrir terminal
4. Ejecutar:
   ```
   npx prisma migrate deploy
   npx prisma db seed
   ```

O usar el botón "Run Command" en Vercel dashboard.
